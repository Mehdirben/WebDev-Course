# WebDev-Course

## Description
A comprehensive web development course covering HTML5, CSS3, and JavaScript fundamentals. This repository contains structured learning materials, interactive exercises, and practical solutions designed for both beginners and intermediate web developers.

## Features
- **Comprehensive Course Materials**: Detailed PDF guides for HTML, CSS, and JavaScript
- **Interactive Learning**: Toggle functionality to show/hide course content and solutions
- **Practical Exercises**: Structured exercises with complete solutions
- **Responsive Design**: Mobile-friendly interface for learning on any device
- **PDF Integration**: View course materials directly in the browser with fullscreen option

## Course Structure
The course is organized into three main sections:

1. **HTML** - Fundamentals of document structure and semantic markup
   - HTML5 basics (TP1)
   - Advanced HTML5 features (TP2)
   - Forms, tables, and semantic elements

2. **CSS** - Styling and layout techniques
   - Basic styling concepts (TP3)
   - Form styling
   - Portfolio design
   - Card layouts

3. **JavaScript** - Interactive programming
   - Core JavaScript concepts (TP4)
   - DOM manipulation
   - Interactive form validation (TP5)
   - Game development

## Technologies
- **HTML5**: Semantic markup, forms, and document structure
- **CSS3**: Modern styling, animations, and responsive design
- **JavaScript**: DOM manipulation, event handling, and interactive features
- **PDF Integration**: Embedded documentation for in-depth learning

## Getting Started
```bash
# Clone the repository
git clone https://github.com/Mehdirben/WebDev-Course.git

# Navigate to the project directory
cd WebDev-Course

# Open the main page in your browser
# (Windows)
start index.html
# (macOS)
open index.html
# (Linux)
xdg-open index.html
```

## Usage
1. Open `index.html` in your browser to access the main course dashboard
2. Navigate to specific course sections using the top navigation
3. Click on "Show" buttons to reveal course content and exercises
4. Use the PDF viewer controls to view course materials in fullscreen or download them
5. Access exercise solutions by clicking "Show Solution" buttons

## Exercise Solutions
Each course section contains practical exercises with detailed solutions:

- **HTML Exercises**: Basic structure, navigation, tables, forms, and semantic HTML
- **CSS Exercises**: Styling forms, creating portfolio layouts, designing cards
- **JavaScript Exercises**: Mathematical operations, DOM manipulation, interactive features
