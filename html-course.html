1. HTML Fundamentals

Introduction to HTML

HTML (HyperText Markup Language) is the backbone of web pages, providing structure and content.

Document Structure:

  • DOCTYPE declaration
  • HTML element
  • Head section
  • Body section

Basic Syntax:

  • Opening and closing tags: <tag></tag>
  • Self-closing tags: <tag />
  • Elements with attributes: <tag attribute="value">

Exercise: Create Your First HTML Page

Create a basic HTML page with a title, heading, and paragraph.


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My Website</h1>
    <p>This is my first HTML page. I'm learning web development!</p>
</body>
</html>

Lists:

3. Links and Images

Links (Anchor Tags):

  • Basic link syntax
  • Relative vs absolute URLs
  • Link to sections within page
  • Link attributes (target, rel)

Images:

  • Image tag and attributes
  • Alt text for accessibility
  • Image formats (JPG, PNG, SVG)
  • Responsive images

Exercise: Create a Photo Gallery

Create a simple photo gallery with linked images.


<h1>My Photo Gallery</h1>

<div class="gallery">
    <a href="large-photo1.jpg" target="_blank">
        <img src="thumbnail1.jpg" alt="Mountain landscape" width="300">
    </a>
    
    <a href="large-photo2.jpg" target="_blank">
        <img src="thumbnail2.jpg" alt="Ocean sunset" width="300">
    </a>
    
    <figure>
        <img src="photo3.jpg" alt="Forest path" width="300">
        <figcaption>A peaceful forest path</figcaption>
    </figure>
</div>

4. Tables and Forms

Tables:

  • Basic table structure (table, tr, td)
  • Table headers (th)
  • Spanning columns and rows
  • Table sections (thead, tbody, tfoot)

Forms:

  • Form elements and attributes
  • Input types
  • Labels and fieldsets
  • Form validation

Exercise: Create a Registration Form

Create a registration form with various input types and validation.


<form action="/register" method="post">
    <fieldset>
        <legend>Personal Information</legend>
        
        <label for="name">Full Name:</label>
        <input type="text" id="name" name="name" required>
        
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>
        
        <label for="password">Password:</label>
        <input type="password" id="password" name="password" 
               minlength="8" required>
        
        <label for="birthdate">Birth Date:</label>
        <input type="date" id="birthdate" name="birthdate">
        
        <fieldset>
            <legend>Interests</legend>
            <input type="checkbox" id="coding" name="interests[]">
            <label for="coding">Coding</label>
            
            <input type="checkbox" id="design" name="interests[]">
            <label for="design">Design</label>
        </fieldset>
        
        <button type="submit">Register</button>
    </fieldset>
</form>

5. Semantic HTML

Semantic Elements:

  • Header and footer
  • Navigation
  • Main and article
  • Section and aside
  • Figure and figcaption

Benefits:

  • Better accessibility
  • Improved SEO
  • Clearer code structure
  • Easier maintenance

Exercise: Create a Blog Layout

Create a blog layout using semantic HTML elements.


<header>
    <h1>My Tech Blog</h1>
    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">Articles</a></li>
            <li><a href="#">About</a></li>
        </ul>
    </nav>
</header>

<main>
    <article>
        <header>
            <h2>Getting Started with HTML5</h2>
            <time datetime="2024-01-19">January 19, 2024</time>
        </header>
        
        <section>
            <p>Article content goes here...</p>
        </section>
        
        <footer>
            <p>Author: Jane Doe</p>
        </footer>
    </article>
    
    <aside>
        <h3>Recent Posts</h3>
        <ul>
            <li><a href="#">CSS Tips</a></li>
            <li><a href="#">JavaScript Basics</a></li>
        </ul>
    </aside>
</main>

<footer>
    <p>© 2024 My Tech Blog</p>
</footer>
