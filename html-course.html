TP1: HTML5 Basics

Exercise 1: Basic HTML Elements

Create a webpage about web technologies with the following structure:

Requirements:
  1. Main Title:
    • Use <h1> for "Technologies of Web"
  2. Section Headings:
    • Create four <h2> headings: HTML5, CSS3, JavaScript, PHP
    • Under each heading:
      • Add a representative technology image
      • Write a descriptive paragraph
      • Use <strong> for important terms
      • Use <em> for language names
  3. Metadata:
    • Add proper charset declaration
    • Include page description meta tag
  4. Documentation:
    • Add HTML comments explaining the structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="description" content="Web Technologies Overview">
    <title>Web Technologies</title>
</head>
<body>
    <h1>Technologies of Web</h1>
    
    <h2>HTML5</h2>
    <img src="html5.png" alt="HTML5 Logo">
    <p><em>HTML5</em> is used for <strong>structuring</strong> and 
    <strong>presenting</strong> content on the web.</p>

    <!-- Repeat similar structure for CSS3, JavaScript, and PHP sections -->
</body>
</html>
Exercise 2: Navigation and Lists

Create a website for the Euro-Mediterranean University of Fès:

Page Structure:
  1. Navigation Menu:
    • Create an unordered list with links:
    • Accueil, Université, Admission, Contact
  2. University Section:
    • Title: "Formation"
    • Ordered list of three programs with descriptions:
      • Engineering and Architecture
      • Humanities and Social Sciences
      • Health Sciences
  3. Admission Section:
    • Process description
    • Unordered list of admission steps
    • Link to official admissions page
  4. Contact Section:
    • Contact information
    • Social media links

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="description" content="Euro-Mediterranean University of Fès">
    <title>Euro-Mediterranean University of Fès</title>
</head>
<body>
    <nav>
        <ul>
            <li><a href="#">Accueil</a></li>
            <li><a href="#">Université</a></li>
            <li><a href="#">Admission</a></li>
            <li><a href="#">Contact</a></li>
        </ul>
    </nav>

    <section>
        <h2>Formation</h2>
        <ol>
            <li>Engineering and Architecture</li>
            <li>Humanities and Social Sciences</li>
            <li>Health Sciences</li>
        </ol>
    </section>

    <section>
        <h2>Admission</h2>
        <p>Process description</p>
        <ul>
            <li>Step 1</li>
            <li>Step 2</li>
            <li>Step 3</li>
        </ul>
        <a href="#">Official admissions page</a>
    </section>

    <section>
        <h2>Contact</h2>
        <p>Contact information</p>
        <ul>
            <li><a href="#">Twitter</a></li>
            <li><a href="#">LinkedIn</a></li>

</body>
</html>
Exercise 3: Course Schedule Table

Create a course schedule for Applied Computer Science:

Table Requirements:
  • Structure:
    • Days: Monday to Friday (columns)
    • Times: 8:00 - 18:00 (rows)
    • Include lunch breaks (12:00-14:00)
  • Course Distribution:
    • Use colspan for multi-hour sessions
    • Use rowspan for extended courses
    • Include all specified courses with proper timing

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="description" content="Course Schedule for Applied Computer Science">
    <title>Course Schedule</title>
</head>
<body>
    <table>
        <thead>
            <tr>
                <th>Time</th>
                <th>Monday</th>
                <th>Tuesday</th>
                <th>Wednesday</th>
                <th>Thursday</th>
                <th>Friday</th>
            </tr>
        </thead>

TP2: Advanced HTML5

Exercise 1: Registration Form

Create a comprehensive registration form with:

Form Elements:
  • Personal Information:
    • Name and surname (text, required, letters only)
    • Birth date (date picker)
    • Email and website (with validation)
    • Password field
    • Gender selection (radio buttons)
    • Country selection (grouped by continent)
  • Interests Section:
    • Hobby checkboxes (Sport, Music, Travel)
    • Additional comments textarea
  • File Upload:
    • File input component
    • Hidden field for max file size

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="description" content="Registration Form">
    <title>Registration Form</title>
</head>
<body>
    <form>
        <fieldset>
            <legend>Personal Information</legend>
            <label for="name">Name:</label>
            <input type="text" id="name" name="name" required pattern="[A-Za-z]+">
            <label for="surname">Surname:</label>
            <input type="text" id="surname" name="surname" required pattern="[A-Za-z]+">
            <label for="birthdate">Birth Date:</label>
            <input type="date" id="birthdate" name="birthdate">
            <label for="email">Email:</label>
            <input type="email" id="email" name="email">
            <label for="website">Website:</label>
            <input type="url" id="website" name="website">
            <label for="password">Password:</label>
            <input type="password" id="password" name="password">
            <label>Gender:</label>
            <input type="radio" id="male" name="gender" value="male">
            <label for="male">Male</label>
            <input type="radio" id="female" name="gender" value="female">
            <label for="female">Female</label>
            <label for="country">Country:</label>
            <select id="country" name="country">
                <optgroup label="Africa">
                    <option value="morocco">Morocco</option>
                    <option value="egypt">Egypt</option>
                </optgroup>
                <optgroup label="Europe">
                    <option value="france">France</option>
                    <option value="germany">Germany</option>
                </optgroup>
            </select>
        </fieldset>

        <fieldset>
            <legend>Interests</legend>
            <label>Hobbies:</label>
            <input type="checkbox" id="sport" name="hobbies" value="sport">
            <label for="sport">Sport</label>
            <input type="checkbox" id="music" name="hobbies" value="music">
            <label for="music">Music</label>
            <input type="checkbox" id="travel" name="hobbies" value="travel">
            <label for="travel">Travel</label>
            <label for="comments">Additional Comments:</label>
            <textarea id="comments" name="comments"></textarea>
        </fieldset>

        <fieldset>
            <legend>File Upload</legend>
            <label for="file">Upload File:</label>
            <input type="file" id="file" name="file">
            <input type="hidden" name="MAX_FILE_SIZE" value="30000">
        </fieldset>

        <button type="submit">Submit</button>
    </form>
</body>
</html>
Exercise 2: Semantic Structure

Build a conference website using semantic HTML5:

Page Sections:
  • Header (<header>):
    • Conference name and dates
    • Main banner content
  • Navigation (<nav>):
    • Main menu structure
    • Section links
  • Sidebar (<aside>):
    • Conference summary
    • Sponsor information
  • Main Content (<main>):
    • Conference program details
    • Embedded video presentation
  • Footer (<footer>):
    • Organizer contact details
    • Social media links

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="description" content="Conference Website">
    <title>Conference Website</title>
</head>
<body>
    <header>
        <h1>Conference Name</h1>
        <p>Dates: January 1-3, 2024</p>
    </header>

    <nav>
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">Program</a></li>
            <li><a href="#">Speakers</a></li>
            <li><a href="#">Sponsors</a></li>
            <li><a href="#">Contact</a></li>
        </ul>
    </nav>

    <aside>
        <h2>Conference Summary</h2>
        <p>Brief summary of the conference.</p>
        <h2>Sponsors</h2>
        <p>Information about sponsors.</p>
    </aside>

    <main>
        <h2>Program Details</h2>
        <p>Details of the conference program.</p>
        <video controls>
            <source src="presentation.mp4" type="video/mp4">
            Your browser does not support the video tag.
        </video>
    </main>

    <footer>
        <p>Contact: info@conference.com</p>
        <ul>
            <li><a href="#">Twitter</a></li>
            <li><a href="#">LinkedIn</a></li>
            <li><a href="#">GitHub</a></li>
        </ul>
    </footer>
</body>
</html>
